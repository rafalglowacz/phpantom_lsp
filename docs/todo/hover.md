# PHPantom — Hover: Improvement Plan

All planned hover improvements have been implemented:

- **Member origin indicators** — hovering a method, property, or constant
  shows whether it overrides a parent, implements an interface, or is a
  virtual (synthesized) member.
- **Enum case listing** — hovering an enum name shows all cases inside
  the code block.
- **Trait method signatures** — hovering a trait name shows public
  members as a scannable summary inside the code block.

No further hover work is planned at this time. New items will be added
here if gaps are discovered.
